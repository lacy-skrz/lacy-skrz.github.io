Web Development & Coding Specialization

I am no stranger to self-teaching necessary skills and finding resources to help me along the way. When I decided to learn more about web development after obtaining the IAAP CPACC certification, I came across the Web Development & Coding Specialization at Coursera. I started taking the courses for this specialization starting in October 2020 with the goal to build a portfolio website, the one you are reading right now, by the end of the year.

One day I was troubleshooting an issue in a knowledge base when I came across a pseudo-element and had an AH-HA moment when I was able to bridge work knowledge with web development concepts. This course gave me the confidence to demonstrate my skills while humbling me to the fact I have much more to learn.

Learning Experience

Throughout this project, the following were my favorite things to research and implement:

  • CSS layout methods with a mobile-first responsive design approach
  • Dark and light mode toggle with a contrast-compliant color palette
  • Media queries for animation and adaptations for reducing motion
  • Local storage to remember user preferences
  • Better document management using version control
